HTML5 Canvas Demo

03 / 2013

Role: Creator

I was building a flexible, depth-sensing touch display for Queensâ€™ Human Media Lab when a friend gifted me a mini Buddha Board. These boards are made of paper that turns black when wet; painting with water on the canvas appears much like sumi ink would on rice paper. As the water evaporates, strokes thin and fade, eventually restoring the canvas to its original blank state.

I decided that a graffiti wall following the same premise would make for an interesting installation, so began implementing it for the large depth-sensing display. I created a prototype using the HTML5 <canvas> element and a pressure-sensitive Wacom Intuos2 tablet. In the demo, pressure dictates stroke weight as it would on the display wall. Thin strokes fade quickly, and heavy strokes fade slowly.

This demo uses proprietary Wacom plugins which can be found here. As such, a Wacom tablet is required for this demo.
