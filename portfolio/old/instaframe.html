Instaframe

Web App

03 / 2013

Role: Creator

Instaframe is a web app designed for quick searches of recent Instagram photos. Along with Peek, I developed this as an exploration of the CSS background-attachment property.

Instaframe is written in HTML, CSS, and JavaScript, and makes use of Instafeed.js. Interesting features of the app include a pure CSS lightbox, an icon font constructed using IcoMoon, and a pure JavaScript text-entry blocking system.

The demo is very buggy, and will likely remain so as it was simply a proof-of-concept. The demo can be viewed below; a recent build of Firefox is suggested.
