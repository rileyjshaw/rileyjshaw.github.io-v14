Peek

Tumblr Theme

03 / 2013

Canary

Role: Creator

Peek is an in-progress Tumblr theme for visual content, born from experiments with the CSS background-attachment property. It was written in pure HTML, CSS, and JavaScript. Much time was taken to ensure that the script would run anywhere, including legacy browsers. If time had time been more of a concern, jQuery could have expedited this process.

Several high-level interface bugs must be cleared up before publishing the theme. For example, photo titles are currently displayed on a hover event; an alternative method must be provided for mobile users. The theme is not entirely cross-browser compatible at this point. A recent build of Firefox is suggested.

At publish date, this theme will offer support for static and animated images, along with infinite scrolling pagination. A planned update will have support for embedded video.
